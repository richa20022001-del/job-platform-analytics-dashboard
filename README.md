## 🚀 Project Overview

This project analyzes user behavior on a job platform using a funnel approach and A/B testing.

The goal is to identify where users drop off and evaluate which experiment group performs better.

---

## 📌 Key Features

### 🔹 User Funnel Analysis

* Jobs Viewed → Jobs Clicked → Applications
* Identifies drop-offs in user journey

### 🔹 Conversion Metrics

* Overall Conversion Rate: **12.10%**
* Conversion Rate A: **11.89%**
* Conversion Rate B: **12.31%**

### 🔹 A/B Testing Insights

* Compared performance of Experiment Group A vs B
* Identified better-performing variant

---

## 📊 Dashboard Highlights

## 💡 Key Insights

* Significant drop from **Views → Clicks (~55%)**
* Strong conversion from **Clicks → Applications (~26%)**
* Experiment B outperforms A → better user experience
